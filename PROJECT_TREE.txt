================================================================================
                            EXECUTION WORKFLOW
================================================================================

    1. VALIDATE SETUP
        python validate_setup.py
        └─→ Checks all dependencies and files
    
    2. RUN ALL ANALYSES
        python run_all.py
        └─→ Executes 5 modules:
            ├─ Data Preprocessing      (2-5 min)
            ├─ EDA Analysis            (1-2 min)
            ├─ Customer Segmentation   (2-3 min)
            ├─ Market Basket Analysis  (3-5 min)
            └─ CLV Analysis            (1-2 min)
    
    3. LAUNCH DASHBOARD
        streamlit run app.py
        └─→ Opens at http://localhost:8501


================================================================================
                         KEY TECHNICAL COMPONENTS

DATA PREPROCESSING
   ├─ Remove canceled invoices

   └─ Engineer temporal features

EXPLORATORY DATA ANALYSIS
   ├─ Revenue trends over time

   ├─ Association rule mining
   └─ Product bundling recommendations

CUSTOMER LIFETIME VALUE
   ├─ Average Order Value (AOV)
   ├─ Purchase Frequency calculation
   ├─ Customer lifespan modeling
   ├─ CLV = AOV × Frequency × Lifespan
   └─ Customer ranking & categorization


================================================================================
                           BUSINESS INSIGHTS
================================================================================

CUSTOMER SEGMENTS DISCOVERED
   ├─ Champions: Recent, frequent, high-value buyers
   ├─ Loyal Customers: High frequency and spending
   ├─ Potential Loyalists: Recent buyers, moderate value
   ├─ At Risk: High value but haven't purchased recently
   └─ Lost Customers: Long time since last purchase

KEY FINDINGS
   ├─ Top 20% customers contribute 60-80% of CLV
   ├─ Clear seasonal patterns in sales
   ├─ Strong product associations for bundling
   └─ Geographic revenue concentration

STRATEGIC RECOMMENDATIONS
   ├─ VIP programs for high-CLV customers
   ├─ Re-engagement campaigns for at-risk segments
   ├─ Product bundling based on association rules
   └─ Seasonal inventory optimization
